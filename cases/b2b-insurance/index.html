Исходная ситуация

B2B-компания по страхованию грузоперевозок по России и миру. Целевая аудитория — логистические компании, которым важны качество услуг и надёжность, а не только цена. Длинный цикл сделки, сложная квалификация лидов, большая часть конверсий происходит офлайн.

Задача

Привлечь целевых клиентов и выстроить аналитику, позволяющую оптимизировать рекламу не по лидам, а по реальным сделкам.

Что было сделано
  • Постоянная доработка сайта для роста конверсии целевой аудитории
  • Подключена сквозная аналитика с передачей офлайн-конверсий из CRM
  • Разработаны кампании в Яндекс.Директ по всем направлениям бизнеса
  • Тестирование гипотез по разным сегментам целевой аудитории
  • Тестирование посадочных, включая квиз и промо-страницы
  • Настройка оптимизации кампаний по качественным лидам и сделкам
