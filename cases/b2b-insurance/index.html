Исходная ситуация

B2B-компания по страхованию грузоперевозок по России и миру. Целевая аудитория — логистические компании, которым важны надёжность и качество, а не только цена. Длинный цикл сделки, сложная квалификация лидов, большая часть конверсий происходит офлайн.

Задача

Привлечь целевых клиентов по страхованию грузоперевозок и выстроить аналитику, позволяющую оптимизировать рекламу не по лидам, а по реальным сделкам.

Что было сделано
  • Постоянная доработка сайта для роста конверсии целевой аудитории
  • Подключена сквозная аналитика с передачей офлайн-конверсий из CRM
  • Разработаны кампании в Яндекс.Директ по всем направлениям бизнеса
  • Тестирование гипотез по разным сегментам целевой аудитории
  • Тестирование посадочных страниц: лендинги, квизы, промо-страницы
  • Настройка оптимизации кампаний по качественным лидам и сделкам
  • Регулярная чистка нецелевых запросов и аудиторий
